A modern, responsive personal portfolio website showcasing projects, skills, and services built with React and Vite.

## Overview

This portfolio is a complete personal website designed to showcase professional work, skills, and services. It features a clean, modern design with smooth animations and a fully responsive layout that works seamlessly across all devices.

---

## Features

**Key Features:**
- Modern, responsive design
- Fast performance with Vite
- Smooth scrolling navigation
- Mobile-friendly interface
- Loading animations
- Project showcase section
- Skills display
- Services offerings
- Functional contact form with EmailJS integration
- Scroll-to-top functionality
- Elegant UI/UX

---

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: CSS3
- **Language**: JavaScript (ES6+)
- **Linter**: ESLint
- **Email Service**: EmailJS
- **Deployment**: Vercel

## Installation

1. **Clone the repository** (or extract the project files)
   ```bash
   git clone https://github.com/Nebiyu14/portfolio
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
